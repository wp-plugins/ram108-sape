SAPE.RU: Adds widget and shortcode to integrate sape.ru contextual links for russian webmasters.

== Description ==

This plugin is in russian language only.

= Плагин SAPE Links для веб-мастеров =

Легкий способ добавить `sape.ru` на свой сайт. Гибкие настройки плагина. Вывод ссылок в любом месте сайта. Подробные инструкции по установке и настройке.

= Возможности плагина =

* Виджет для размещения ссылок в области виджетов сайта. Используйте несколько копий виджета, чтобы отобразить ссылки в разных частях сайта, выберите, какое количество ссылок размещать.

* Макрос для отображения ссылок на страницах сайта. Использование: `[sape]` для вывода всех ссылок, `[sape count=3]` с указанием количества ссылок в параметре `count`. 

* Автоматические контекстные ссылки в публикациях сайта при включенных настройках плагина.

* Вывод ссылок в теме оформления. Вставьте код `do_shortcode('[sape count=3');` там, где вы желаете размещать ссылки. 

= Системные требования =

* [Требуется регистрация](http://www.sape.ru/r.zPvHCvTdOj.php "Перейти на сайт sape") на сайте sape.ru
* Работает с php 5.3 и выше

= Благодарность =

== Installation ==

1. Загрузите папку плагина в директорий `/wp-content/plugins/`
1. Активируйте плагин в меню 'Плагины'
1. Перейдите на страницу настроек плагина SAPE Links и следуйте инструкциям

Хороших вам заработков!

== Frequently Asked Questions ==

= Что такое sape.ru =

Sape.ru — это интеллектуальная система платного размещения ссылок для веб-мастеров. Благодаря гибким настройкам, система автоматически подбирает подходящие ссылки, которые вы вправе принять или отклонить для размещения на вашем сайте.

= Могу ли я зарабатывать на своем сайте? =

Пройдя простую [регистрацию в системе](http://www.sape.ru/r.zPvHCvTdOj.php "Перейти на сайт sape"), ваш сайт может приносить стабильный доход.
