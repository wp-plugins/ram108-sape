SAPE.RU ссылки для веб-мастеров. Добавляет виджет, макрос [sape] и контекстные ссылки на страницы сайта.

== Description ==

This plugin is in Russian language only.

= Плагин SAPE Links для веб-мастеров =

[Sape.ru](http://www.sape.ru/r.zPvHCvTdOj.php "Перейти на сайт sape") — интеллектуальная система платного размещения ссылок на своем сайте. Гибкие настройки плагина. Вывод ссылок в любом месте сайта. Подробные инструкции на русском языке. 

= Возможности плагина =

* Виджет для размещения ссылок в области виджетов сайта. Используйте несколько копий виджета, чтобы отобразить ссылки в разных частях сайта, выберите, какое количество ссылок размещать.

* Макрос для отображения ссылок на страницах сайта. Использование: `[sape]` для вывода всех ссылок, `[sape count=3]` с указанием количества ссылок. 

* Автоматические контекстные ссылки в публикациях сайта при включенных настройках плагина.

* Вывод ссылок в теме оформления. Вставьте код `<?php do_shortcode('[sape count=3'); ?>` там, где вы желаете размещать ссылки. 

= Системные требования =

* [Требуется регистрация](http://www.sape.ru/r.zPvHCvTdOj.php "Перейти на сайт sape") на сайте sape.ru
* Работает с php 5.0 и выше

== Installation ==

1. Скачайте и разархивируйте плагин.
1. Загрузите папку `ram108-sape` в директорий `/wp-content/plugins/` вашего сайта
1. Активируйте плагин в меню 'Плагины' в панели администратора
1. Перейдите на страницу настроек `[ram108] SAPE Links` и следуйте инструкциям

Стабильных вам доходов!

== Frequently Asked Questions ==

= Что такое sape.ru =

Sape — это интеллектуальная система платного размещения ссылок на своем сайте. Благодаря гибким настройкам, система автоматически подбирает подходящие ссылки, которые вы вправе принять или отклонить.

= Могу ли я зарабатывать на своем сайте? =

Пройдя [простую регистрацию](http://www.sape.ru/r.zPvHCvTdOj.php "Перейти на сайт sape") в системе sape, ваш сайт может начать зарабатывать уже сейчас.

= Рекомендации по размещению ссылок =

Разнесите ссылки по разным частям сайта, старайтесь установить ссылки "повыше" на странице. Оптимальное количество ссылок на странице — не более трех.

Активируйте виджет SAPE Links в боковой колонке (в начале и в конце) и в подвале сайта. Укажите, какое количество ссылок выводить. Например, '1' ссылка в начале колонки, '1' ссылка в конце колонки и 'Все' в подвале. 

= Почему все ссылки отображаются в одном месте =

"Все" ссылки выводятся там, где в параметрах указано 'Все' (или в макросе `[sape]` без указания количества ссылок). Если все ссылки выведены, то остальные виджеты остаются пустые. Только последний виджет должен иметь параметр 'Все' для отображения оставшихся ссылок. 

= Плагины кэширования WP Super Cache, W3 Total Cache, Hyper Cache и т.п. =

Установите сброс кэша каждые 6 часов, чтобы новые ссылки sape появлялись на вашем сайте. Во время первичной установки плагина SAPE Links и добавления сайта в систему sape рекомендуется отключить плагин кэширования.
