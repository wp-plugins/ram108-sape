This plugin is in Russian language only.

= Плагин SAPE Links для веб-мастеров =

[Sape.ru](http://www.sape.ru/r.zPvHCvTdOj.php "Перейти на сайт sape") — интеллектуальная система платного размещения ссылок. Легкие настройки плагина, вывод ссылок в любом месте сайта.

= Возможности плагина =

* Виджет для размещения ссылок. Используйте несколько копий виджета для отображения ссылок в разных частях сайта. Выберите, какое количество ссылок выводить.

* Макрос для отображения ссылок. Использование: `[sape]` для вывода всех ссылок, `[sape count=3]` с указанием количества ссылок, вывод в виде блока `[sape block=1]`.

* Вывод ссылок в теме оформления. Используйте код `<?php echo do_shortcode('[sape count=3]'); ?>` там, где вы желаете разместить ссылки.

* Автоматические контекстные ссылки в публикациях сайта при включенных настройках плагина.

* Вывод счетчика sape отдельно от ссылок в подвале сайта. Скрытие виджета со страницы при отсутствии ссылок.

= Системные требования =

* [Требуется регистрация](http://www.sape.ru/r.zPvHCvTdOj.php "Перейти на сайт sape") на сайте sape.ru
* Работает с php 5.0 и выше

== Frequently Asked Questions ==

= Что такое sape.ru =

Sape — это интеллектуальная система платного размещения ссылок на своем сайте. Благодаря гибким настройкам, система автоматически подбирает подходящие ссылки, которые вы вправе принять или отклонить.

= Как я могу заработать на своем сайте? =

Пройдя [простую регистрацию](http://www.sape.ru/r.zPvHCvTdOj.php "Перейти на сайт sape") в системе sape, ваш сайт может начать зарабатывать уже сейчас.

= Рекомендации по размещению ссылок =

Разнесите ссылки по разным частям сайта, старайтесь установить ссылки "повыше" на странице. Оптимальное количество ссылок на странице — не более трех.

Активируйте виджет SAPE Links в боковой колонке (в начале и в конце) и в подвале сайта. Укажите, какое количество ссылок выводить. Например, '1' ссылка в начале колонки, '1' ссылка в конце колонки и 'Все' в подвале.

= Плагины кэширования WP Super Cache, W3 Total Cache, Hyper Cache и т.п. =

Установите сброс кэша каждые 4-6 часов, чтобы новые ссылки sape появлялись на вашем сайте. Во время первичной установки плагина SAPE Links и добавления сайта в систему sape рекомендуется отключить плагин кэширования.

= Я установил плагин, но система не видит код на сайте =

Для первичной установки кода sape и прохождения модерации сайта, необходимо, чтобы виджет SAPE Links был установлен НА ГЛАВНОЙ странице сайта.

= Почему все ссылки отображаются в одном месте =

"Все" ссылки выводятся там, где в параметрах указано 'Все' (или в макросе `[sape]` без указания количества ссылок). Если все ссылки выведены, то остальные виджеты остаются пустые. Только последний виджет должен иметь параметр 'Все' для отображения оставшихся ссылок.

= Отключить wptexturize =

Фильтр wptexturize обрабатывает исходный текст сайта, заменяя некоторые пунктуационные знаки (например, тире) на спецсимволы. В результате контекстные ссылки, содержащие эти знаки, могут быть не распознаны системой sape и отмечены статусом ERROR. Если подобная проблема наблюдается на вашем сайте, отключите wptexturize в настройках плагина. В противном случае, не стоит отключать wptexturize.
